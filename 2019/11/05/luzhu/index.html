京城卤煮之我见

发表于 2019-11-05 更新于 2022-08-29 分类于 Life 评论： 阅读次数：

卤煮火烧是一道地道的北京小吃，如果你爱吃内脏，那它必定是你不能错过的美味; 反之则是吾之蜜糖，彼之砒霜，唯恐避之不及。其实许多北京小吃还是挺好吃的，只是术业有专攻，吃每样小吃得找对店铺，自然得花些功夫。加之街头许多店挂羊头卖狗肉，打着名吃的旗号却做不出那个味儿，大部分人尤其是游客只想尝个鲜，随口一吃就下了结论“这东西真难吃”，殊不知只是没找对地方。而许多景点又有太多所谓小吃一条街，表面看来方便了食客将美食一“街”打尽，实际却砸了招牌，几无回头客。

先聊聊自己跟卤煮的缘分，记得第一次吃卤煮是在北大农园的食堂里，没吃过就要了一碗尝鲜，发现很合口味，再加上食堂的卤煮不是天天有，于是就想尝下地道的卤煮是什么滋味儿。当时网上搜了下，发现“小肠陈”非常有名，而且这名字听起来就很正宗：符合菜肴+姓氏的老字号风格。

话不多说，哥儿几个找个机会就去新街口的百货大厦找了家小肠陈。时间比较久远，只记得当时四人要了四碗，基本上都没怎么吃，味儿重且冲，汤的颜色偏棕色，不发红也不是酱色，总之实在是来不动。乘兴而至，败兴而归，回来之后哥儿几个还是觉得食堂里经过改良的“卤煮”更好吃。

这家店现已不在了，依我现在的认识，这家分店应该做的是相当不正宗，汤底颜色不正，肠的处理也有些问题，腥味儿太浓。吃卤煮还得到街巷的小店气氛才对，没听说过哪家酒楼的卤煮做得地道的，这家开在百货大楼的店味道一般实属意料之中。

言归正传，介绍下卤煮是怎样一种存在，先来张门框胡同卤煮的图直观感受下：

京城女子卤煮报告，一张图将卤煮是什么解释得很明白，用文字写出来就是：

卤煮火烧 = 菜底 + 火烧 + 汤 菜底 = 肠 + 肺 + 炸豆腐 + 五花肉

明白了这个，就可以理解老店墙上的价格表了，一份卤煮多少钱，加菜底多少钱，更细的会分加小肠、大肠多少钱之类的。

下面具体聊聊每样食材的讲究。先说菜底，卤煮里面最好吃的当然是肠了，肠是一碗卤煮的精髓。我觉得一家店对于卤煮的理解就体现在肠的处理上：洗得干净程度，肠内壁上的油留下多少，卤好的肠的颜色。我不大喜欢吃肺，肺的口感有点绵，不太有嚼劲儿。一般可以在点单的时候说不要肺，讲究的店会替换成一块炸豆腐。炸豆腐没什么特别的感觉，没觉得哪家特别出彩的。五花肉有人称之为卤煮的灵魂，主要源自于“苏造肉”卤煮的历史，不过现在有些店都不加这味料了，我对这块肥肉也没有太多好感。

再说火烧，一块死面饼，注意，如果看到一家店的火烧是发面饼就不用吃了，肯定不正宗。火烧一般是放在老汤的锅沿周围咕嘟，一般以小火咕嘟个3~4小时之后为佳，发面饼撑不了这么久就泡粉了。我见过的操作基本是边捞边放，火烧在老汤锅的锅沿周围咕嘟上一圈后被捞上来切成零形块儿。锅的周长有限，所以食客太多的地方的火烧注定是咕嘟不够时辰的，所以入不了味儿。忘了在哪看了句话，大意就是卤煮不吃人多，就是这个道理。

最后说汤，好汤应该是颜色醇正，看起来很有食欲，可以直接入口喝的。个人觉得比较好的汤是褐中发点红，不能太咸。

端上卤煮后，还有一些小料可以根据口味自行添加：腐乳，韭菜花，生蒜水，香菜。

卤煮去哪吃

简单来说，正宗的卤煮还是得到四九城的老城区里吃，毕竟那些出名的老店都在老城里。如果不生活在老城区里又不愿意跑远尝这口，就可以搜索一下“门框胡同卤煮”这样的连锁店，虽然味道谈不上惊艳，至少也有八成功力，不至于踩雷。下面小结下我吃过的卤煮店。

北新桥卤煮老店

一家老店，谢霆锋在2017年曾光临这家店让它更火了起来。

分为精品卤煮（大小肠）、大肠卤煮和小肠卤煮。不必说，每次必点精品卤煮，大小肠一碗尝尽。北新桥应该是卤煮圣地了，之前听糖蒜广播主播说过吃卤煮就得去北新桥。

下面这张图是师傅从锅里捞出肠、豆腐在案板上切的图片，这家店小师傅有个切的特点就是“甩刀”，姿势潇洒。

这家店应该说是我最喜欢的，卤煮高配，味道如一。精品卤煮的话，一碗的量很大，下肚基本也就到位了。大小肠的处理恰到好处，汤里漂些零星油花而不腻，汤的颜色正，入口咸淡合适。唯一小缺点就是肺的口感一般，可以换成豆腐。几样小料自己加，都调和的比较稀，可以来个几小勺。

此外就是用餐环境有点恶劣，不大的一家小店总是人满为患，有时队能逆时针绕一圈排到门外去，这就可以把自己想像成火烧了，进门排队进汤里被咕嘟，绕一圈点单取卤煮出锅，哈哈。跟人拼桌是常态，有时快吃完了还有端着盘子没座位的同学默默站在桌边等你走人，体验不是很好。不过这东西也不是法餐，用不着细品，即吃即走。接地气儿的店里位置不多，也不宜聊天谈事儿~

小肠陈

这回尝的是南横东街的小肠陈饭庄总店，虽然多年前在新街口吃出了心理阴影，但后来在北新桥卤煮已被正名，所以来总店尝尝手艺。

这家店人也特别多，可能是由于是假期的原因，中午快2点了拼桌都没地方坐。卤煮的量不太大，价格稍便宜些，可以多加份菜底。老店就是老店，汤底色正，味道很好。只是在南城，平时并不太常去，吃的次数有限。

门框胡同百年卤煮

连锁店，平日解馋佳选。这应该是吃过频次最高的店了，因为它满城开店，品控做得还算不错，至少端上来有八成以上的功力。如果是入门的话，可以就近找一家尝试。

门框卤煮的量一般，但一碗吃完也差不多了。个人觉得它的肺是最好吃的，不过菜底中好像没有五花肉。

小街卤煮

小街卤煮味道比较中庸，不过这家有特色卤煮面什么的，没有尝试。

黄寺卤煮

黄寺卤煮位置不大好找，在一个地下美食城里面。店面挺大，酱肘子可能比卤煮还要出名。

卤煮风格比较粗犷，切的块儿略大，肥肠上色较浅，偏黄偏肥，从图中漂着的油花儿就能看出来。汤底的颜色个人不是很欣赏，稍显暗淡，肺的质量不行。小料可以加，不过注意腐乳千万别加多了，没有调稀，多来两勺的话可能就被齁着了……

卤煮吕

一家在平安里特不起眼的小店，墙上写着“京城一绝，酱香卤煮。” 蔡澜也曾光临此店。 从上面照片就能看出小店的风格，感觉卤煮店这种存在就是多年如一的传承，如果把店里翻修得非常现代，可能食客反会觉得缺了那么点意思。

酱香卤煮，这成色偏深，口味也偏重。菜底的内容总体可用“紧致”二字形容：肠油基本没有（据说只用肠头），只剩下一层皮，颜色是深酱油色；火烧刀工不错，也挺有嚼头。汤底也是酱油色，太咸了，不大好直接喝。但这碗卤煮吃下来还是很满足，有特色的一种别样风味。

对于好这口儿的人来说，卤煮确实有着难以抵挡的魅力。过段时间不吃就会想来一碗，不过毕竟全是内脏，出于对健康的考虑，还是不要太过频繁地吃，再说，少食多香不是？
