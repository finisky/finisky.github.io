CS常见读错名词集锦

发表于 2019-11-25 更新于 2022-08-18 分类于 Misc 评论： 阅读次数：

计算机圈中有些常见被中国人读错的单词，现列举部分如下：

AJAX

/'eɪdʒæks/，不读阿假克斯。

ASCII

/'æski/，不读[ask]。

cache

/kæʃ/，不读[catch]。

Git

/ɡɪt/，不读[吉特]/dʒit/。

LaTeX

/'leɪtɛk/，不读/'leɪtɛks/，与TeX一样，X不发音。

Lucene

/'lu:sin/，不读/lu:sən/。

Ngnix

[engine x]/'endʒɪneks/, 不读[engine ks]。

Putty

/'pʌti]/，不读/pu:ti/、/pjuti/。

SQL

/'si:kwəl/，不读/'sə:kəl/。最常见的错读没有之一，甚至DB方向的大学教授也会读错，以至于听到的绝大多数都是错误读音。

TeX

/tɛk/，不读/tɛks/，X不发音。

Ubuntu

/u'buntu:/，不读/ju'buntu:/。

.NET

[dot net]，不读[点net]。
