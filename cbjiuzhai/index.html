十月川北自驾游记——九寨沟

发表于： 2021-11-29 分类于： Life 阅读：≈ 12分钟 浏览： 评论：

本文为十月川北自驾游记系列之四，九寨沟。

十月川北自驾游记系列：

  1. 十月川北自驾游记——成都
  2. 十月川北自驾游记——乐山
  3. 十月川北自驾游记——都江堰
  4. 十月川北自驾游记——九寨沟
  5. 十月川北自驾游记——黄龙

游记之前，先聊两个FAQ：

  • 九寨沟黄龙旅游是否会高反？

九寨沟县城、酒店和沟口海拔在2000米左右，九寨沟景区最高点长海和原始森林约3000米，个人体验3000以下不怎么有高反的感觉。3000以上再运动可能会有些气短，但总体没问题。 黄龙景区就稍高些，入口就3200，最高点3568，如果先玩九寨再玩黄龙的话，基本也就适应了，问题不大。我和父母都是徒步上下山，没坐索道，有些气喘，可以接受。景区路上还设有氧吧，有卖小氧气瓶（200次/瓶）的，我觉得没有必要。一路上看到吸氧的人也不多，也没见到任何人有明显的高反。在这个海拔谈高反，个人观点纯属生意，99%的人没必要担心这个，觉得有高反，大部分属于心理作用。

  • 成都到九寨沟路况如何？

毕竟是旅游线路，路况不错，驾驶难度小于丽江到香格里拉和青甘大环线。 从成都到都江堰是高速，自不必提。然后是G213国道到松潘，G544到九寨沟。两段国道都是双向单车道的山路，有山路驾驶经验的话非常好开。值得一提的是隧道比较多，尤其是在都江堰到汶川段，有隧道群。

自驾前往九寨沟

成都到九寨沟的自驾路线基本就是延着岷江不断上行，走蓉昌高速，G213和G544国道： 成都 -> 都江堰 -> 汶川县 -> 茂县 -> 叠溪 -> 松潘 (川主寺) -> 九寨沟。

从都江堰自驾到九寨沟导航时间约6个半小时，实际开下来加上休息和拍照时间差不多得8小时，也就是得一整天，还是比较辛苦。大部分都在走国道，也会穿过各种村庄，自然就有不同的限速。但还是自驾的常见痛点，很多路段限速未知，只见限速牌，不见限速解除牌，你开车守规矩，但大部分车是不守规矩的，甚至还要在后面喇叭大灯一顿催你超速，劣币驱逐良币。很多大货开得比小车快得多，在这种狭窄的山路上十分危险。路上虽然看着摄像头挺多，但管用的不多，常跑的这些货车司机根本不鸟这些摆设。

随着海拔越来越高，景色也越来越好，自驾的体验还是不错的。实测延途海拔：

这条国道也是旅游大道，全程有手机信号，服务区、洗手间、加油站也不少，相比青甘大环线的几百公里的无人区友好太多。虽然也是盘山路，在山谷中前行，但并没有崎岖坎坷的路，总体来说难度并不高。

我们是10点出发，下午5点左右到了弓杠岭，海拔3600，这里有休息区观景台，还有九寨沟的大牌子，很多车都停下来拍照。这里因为海拔高，温度比较低，得穿着羽绒服下车玩耍，而且5分钟不出手都冻僵了。

弓杠岭的景色不错，远处的群山有雾霭萦绕，仿佛置身世外桃源。

过了弓杠岭有一段长下坡，接着就到了目的地九寨沟。订的酒店离景区大门约3公里，有大的停车场，基本没车，但酒店环境还不错，比预想的要好很多，房间有大城市的3星半水平，想到当年在祁连山县城住的小旅店，很满意了。

一天舟车劳顿，想着晚上吃顿好的，尴尬了，文末再细说。

九寨沟

九寨沟位于四川省西北部岷山山脉南段的阿坝藏族羌族自治州九寨沟县漳扎镇境内，地处岷山南段弓杠岭的东北侧。距离成都市400多千米，系长江水系嘉陵江上游白水江源头的一条大支沟。 九寨沟自然保护区地势南高北低，山谷深切，高差悬殊。北缘九寨沟口海拔仅2000米，中部峰岭均在4000米以上，南缘达4500米以上，主沟长30多公里。

九寨沟的得名来自于景区内九个藏族寨子（树正寨、则查洼寨、黑角寨、荷叶寨、盘亚寨、亚拉寨、尖盘寨、热西寨、郭都寨），这九个寨子又称为“和药九寨”。由于有九个寨子的藏民世代居住于此，故名为“九寨沟”。

九寨沟要玩一天还是两天？这个问题我也纠结了下，说什么的都有。就我自己玩下来的经历来看，一天其实是够的，就是要规划得很好，行程不能有一点耽误。

7点半开门，6点关门，看起来时间很长，真玩起来时间过得很快，原因在于坐车和徒步都很耗时间，拍照打卡速度也提不上去，所以一天时间还是很紧。

如果想玩得轻松些，并且把景点都看一遍，那么两天是合适的。考虑到来九寨沟往返要开两个整天的车，时间成本较高，玩两天感觉更值一些，果断买了两日联票。

强烈建议只玩一天的游客提前做好攻略，至少把九寨沟各景点在什么位置，游览路线做好功课。比如最主要的几个景点要弄明白：

  • 五花海：九寨沟最出名的水景，有九寨沟一绝和九寨精华之美誉。
  • 珍珠滩瀑布：86版西游记取景地。
  • 五彩池：九寨沟最小而颜色最为丰富的池子。

实际上游玩下来的感受是没有照片中的美，道理很简单，现在的景点PS的痕迹都太重了，加上不同天气的景色本身也有差异。

九寨沟景区(第一天)

早上7点多吃了早饭出酒店门一看心凉了半截，漫山遍野的大雾，地上还湿漉漉的，刚下了小雨。想起青城山游览的情形，担心九寨沟玩不好了。后来发现是杞人忧天，与之前去香格里拉和青海一样，高原天气瞬息万变，不必在乎天气预报。过了10点，天突然就放晴了。

九寨沟早上7点半开门，我们大概8点多到达景区。可以看出九寨沟是夹在两山之间：

九寨沟景区是个Y字形，由三条沟构成：树正沟，日则沟，和则查洼沟。景区门口海拔最低，大约2100米，长海海拔最高，3000米。进了景区之后就要坐电瓶车，拉到最远点，即这个Y字形的两个顶端：要么是长海，要么是今年9.28号新开的原始森林。但在上电瓶车的时候并不知道是去哪个点，车会在Y字形的三叉口，也就是诺日朗服务中心处被调度到长海或原始森林，车程大概25分钟。

树正寨

第一天的行程不是很合理，可能因为觉得要玩两天的原因，所以第一天的心态有些过于放松了。

先坐了个区间电瓶车，只从沟口拉到了树正寨，其实并不知道应该怎么玩，来之前只简单看了眼攻略，知道九寨沟的游览是徒步+坐旅游车的形式。车停在树正寨后，工作人员告诉我们可以走过栈道先游览树正寨，徒步到下一站犀牛海坐车再去长海或原始森林。考虑到有两天，我们就比较悠闲地在这里溜达：

徒步沿栈道走到犀牛海乘车点，最喜欢的一张照片，秋叶倒影：

原始森林

排队从犀牛海坐车到原始森林，中间经过诺日朗服务中心，总车程大概35分钟。原始森林就是传说中9月28号在震后刚刚重开的景点，海拔也比较高，3000米，刚上来稍有点憋气的感觉。景色只能说一般，有很多高大的杉树：

我们先是把整个原始森林栈道走了个遍，然后想乘车往回玩。但到了乘车点发现还有两个海子：草海和天鹅海。于是接着沿栈道徒步，走了3、4公里才到这两个海子，途中基本没什么游客，后来想想，大家都去五花海和珍珠滩之类的著名景点了……

但九寨沟给我最惊艳的一景却是在原始森林的剑岩:

熊猫海

第一天玩没经验，到了下车点工作人员就提示你下车，其实是可以继续在车上直接到下一站的，可以按需略过一些次要景点。每个点都玩的主要问题在于体力跟不上，乘车点之间常常有一段距离，全程徒步的话，体力消耗比较大，加上游客人数众多，比较耗时。

五花海

到了五花海已经1点半了，在这里吃完午饭2点多了，珍珠滩和另一侧长海还都没去。

五花海是九寨沟最著名的景点之一，两点多这个时间，光线一般，还有点逆光，但依然难掩其美：

珍珠滩

匆匆看了眼五花海，人太多，也就没怎么留影，留给明天吧。然后徒步走到了珍珠滩瀑布，也就是著名的西游记取景地。

长海

摆渡到Y字形的另一条线的最远端长海也颇费了些周折。没有直达车，必须先回到诺日朗游客中心才能再去长海，结果回到诺日朗后，找不到发往长海的乘车点，只有往诺日朗瀑布方向走的车，也就是回沟口的车。此时也完全找不到工作人员，只好问旁边的商贩，回答只有三个字：“往上走”。一顿折腾，最后发现确实是往上走，要穿过诺日朗服务中心和一条商业街，才到了去长海方向的乘车点。一路上就是看不到明显指示牌，这点做得实在是不好。

长海

长海是九寨沟里最大的海子，不过从栈道这里看长海只能看到其中的一部分。长海是高山湖泊，看起来有些天池的感觉。

从树正寨看远处的高山，与昨天一样，过了中午就晴空万里了。

吐槽

九寨沟的槽点太多了，总结起来主要有三点：景区管理不到位，当地人不够诚信，九寨无美食。

先说景区管理，如果提前不做功课的话，很容易玩得摸不着头脑。第一天游览的时候，常常不知道下一站去哪，也没人可问。基本只能到了车站才能问摆渡车上的工作人员，由于人多能维持车上秩序就不错了，回答问询的态度不好。

从酒店到景区有几公里，即使是自驾过去，也强烈建议打车前去景区。原因很简单，停车难，官方停车场修得离景区大门比较远，而且收费很贵，按小时计价，打车可以直达景区门口，也更划算。

堂堂5A景区，官方停车场不方便，看看私家停车场的收费牌：

据出租车司机说，这里有上万家酒店，可能因为非假期游客较少，我们看到许多店铺都是打烊状态。

再聊聊本地人的诚信问题，说两件小事儿。

一个是打车去景区，师傅故意不打表，下车时收费的原话是“你们三位，每位5块”，头次经历正规出租车按位收费的。另外几次打车，司机在跟你攀谈的过程中一定有目的，比如兜售九寨千古情的票，从他这买更便宜；亦或听说明天你们计划要去黄龙，吓唬你说黄龙一定会高反，可以从他这买票去神仙池，是一样的景点等等。

还有买青稞饼，在饭店里买，5块一个，老板信誓旦旦地说：这是这里的特色，我下午刚打的50个饼，多了都没有。结果出门发现每家超市都有同样包装的青稞饼，问了旁边一家，3块一个。高潮来了，第二天去这家超市买5个，张嘴就是4块一个，坐地起价，因为我们知道昨天的问价，所以老板也就不再坚持。但这几件小事儿能说明一些管理混乱和诚信的缺失，能蒙就蒙。

在九寨沟住了三天，最后聊聊“九寨无美食”。从点评上也能反映出来，4分以上的店都不多。从景区到酒店这三公里除了酒店就是饭店，但搜点评发现炒菜的店不多，主流菜品是藏餐、耗牛肉锅和烤鱼，全都是农家乐水平。

第一晚先吃了个九寨鱼庄，点了条香辣鱼，量是不小，但除了辣味没什么味，而且由于没有菜，吃完总觉得吃得不太巴适。

后两晚找了离沟口比较近的两家川菜馆，都是出景区吃晚饭。第一晚吃得尚可，算是普通川菜馆。第二晚简直了，青椒肉丝的肉感觉不大好；土豆烧排骨，基本全是骨，肉根本咬不动，跟老板还确认了下这肉到底熟没熟，老板拍着胸脯说是中午刚进的肉，中午自己还吃了很多，非常好吃，放心吃。最后来一鸡蛋汤，上来看着像涮锅水，喝一口跟没放盐一样，难吃到连照片都不想拍。

后来发现点评上都搜不到这个重庆川菜馆的名字，再看原来这些店总改名，以前叫“蜀之味川菜馆”，点评上只有3.5分，而且第一个就是差评。这么看来，改名的原因就可以理解了。不太明白为什么不想着提升点菜品质量，总搞这些歪门邪道。可能因为这里天高皇帝远，好吃难吃你也得吃，没有竞争才是根本原因。

九寨沟确实值得一游，美景绝非浪得虚名，但旅游体验就见仁见智了。

离开九寨沟，前往黄龙：

自驾游
