微软与ChatGPT联手会带来什么？

发表于： 2023-01-23 分类于： Product 阅读：≈ 3分钟 浏览： 评论：

最近微软投资ChatGPT的消息甚嚣尘上，二者的联手会给产业和用户带来什么？

# Microsoft in talks to invest $10 bln in ChatGPT-owner OpenAI

从新闻上来看，微软会将ChatGPT集成到Office和Bing Search。但实际情况可能不止于此，微软擅长做平台，CVP已经在 Azure Blog 称ChatGPT将不久在Azure OpenAI Service上可用:

Customers will also be able to access ChatGPT—a fine-tuned version of GPT-3.5 that has been trained and runs inference on Azure AI infrastructure—through Azure OpenAI Service soon.

好消息是这个服务可以直接让中小企业基于API研发产品而无须自行研发模型。坏消息是它的效果太好以至于自己训练的模型不能达到同水平的效果，形成对此底层服务的强依赖。

值得注意的是，像ChatGPT这样的模型可能会形成马太效应，赢者通吃。模型效果很大程度上来源于真实数据，如果有一个玩家占据了流量入口，那么通过线上不断地与用户交互形成正向闭环，将会越来越强，从而形成一定的壁垒。

ChatGPT与微软的联手可能会催生很多有趣的产品。微软在之前也有过AI助手，比如已经不做的Cortana，是否可以考虑重回市场？基于ChatGPT出色的代码生成能力，将其集成到Visual Studio和VS Code中则是一个特别直接的应用场景，显著提升生产效率。再如使用ChatGPT给各种游戏的NPC赋予对话能力，这也是最近各大游戏公司与元宇宙公司都在主力投入的方向。

ChatGPT与Office的结合会使Office套件会变得更加智能：

  • Outlook: Google在几年前推出的SmartReply技术，可以帮助用户快速生成邮件回复。有了ChatGPT，Outlook生成回复的质量会大幅提升。
  • Word: 更好地帮用户识别语法错误，根据上下文生成文章或报告的草稿，甚至可以写小说进行文学创作。
  • PowerPoint: 以前只是推荐部分设计模板，现在可以帮你生成完整的PPT，并根据用户的要求不断修正。

ChatGPT与Bing的结合则是相辅相成，一方面有搜索技术的加持，可以让ChatGPT基于搜索结果进行参考回复；另一方面，用户与搜索引擎的交互也可能发生一定的变化。经过多年搜索引擎的反向训练，用户已经习惯了关键词检索。但让用户用自然语言进行搜索和交互的探索似乎从未停止，现在有了ChatGPT，用户的信息检索是否可以通过对话完成？或者说提供的搜索结果经模型加工后更像真人的回答。

# Microsoft to challenge Google by integrating ChatGPT with Bing search

By using the technology behind ChatGPT — which is built by AI company OpenAI — Bing could provide more humanlike answers to questions instead of just links to information. Both Google and Bing already surface relevant information from links at the top of many search queries, but Google’s knowledge panels are particularly widespread when it comes to searching for information about people, places, organizations, and things.

不过商业化要考虑的问题不止这些，ChatGPT的产品落地并不简单。第一，模型的安全性与正确性问题。ChatGPT毕竟是个黑盒的神经网络，再智能也有一定的错误率。模型虽然经过安全性和防诱导的训练，但人类语言博大精深，依然很难保证模型回复的100%安全与回答的逻辑正确。第二，成本问题，线上部署大模型并经受流量考验是个烧钱的活儿，这一点可以从Azure发布的服务收费上体现，成本足够低商业模式才能成立。

总之，微软与ChatGPT的联手会给行业带来新鲜血液和更多可能性，同时也会催生更多有趣产品的出现，期待它们能影响和改变我们的生活。
